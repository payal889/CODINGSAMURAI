# Movie Recommendation System

## Objective
The objective of this project is to recommend movies based on user ratings and genre preference using Python.

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
The dataset used in this project is taken from Kaggle.
MovieLens Dataset (Small Version)

Source: https://www.kaggle.com/

The dataset contains the following files:
- movies.csv
- ratings.csv

## Project Description
In this project, movie rating data is analyzed to find the most popular and highly rated movies.
Average ratings are calculated for each movie.
Genre-wise movie distribution is visualized using bar charts.
Based on genre preference, top-rated movies are recommended to the user.

## Features
- Calculates average movie ratings
- Displays top-rated movies
- Shows genre-wise movie count
- Recommends movies based on selected genre

## How to Run the Project
1. Open the `.ipynb` file in Google Colab
2. Upload `movies.csv` and `ratings.csv`
3. Run all the cells sequentially

## Conclusion
This project demonstrates basic data analysis and recommendation logic using Python.
It is suitable for beginners learning data analysis.
