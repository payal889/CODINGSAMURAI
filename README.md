# CODINGSAMURAI
# Movie-recommendation System using python


## Objective
To recommend top-rated movies based on user ratings and genre.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
MovieLens (small version)

## Features
- Calculates average movie ratings
- Shows genre-wise movie distribution
- Recommends top movies by genre

## How to Run
Open the .ipynb file in Google Colab and run all cells.
