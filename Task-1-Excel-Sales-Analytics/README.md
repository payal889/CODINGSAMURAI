
# Task 1: Sales Analytics (Excel)

## Objective
To analyze sales data and create a basic sales dashboard using Microsoft Excel.

## Dataset
The dataset includes the following columns:
- Date
- Product
- Quantity
- Price
- Total Sales

## Tools Used
- Microsoft Excel
- Pivot Tables
- Bar Chart
- Line Chart

## Key Analysis
- Calculated total sales using formulas
- Created daily sales trend using line chart
- Identified top-selling products using pivot table and bar chart

## Outcome
This project provides a clear understanding of sales performance and product-wise contribution.
